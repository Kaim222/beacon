# BEACON

**Budget & Expenditure Accountability CONnector — Massachusetts**

Independent analysis of $98.3B in Massachusetts state spending, cross-referenced with campaign finance records, lobbyist registrations, and federal PAC disclosures.

**What this tracks:**
- $98.3B in state expenditures (FY2025)
- $27.5B in unclassified payments (28% of all spending)
- 8 lobbying firms, 349 clients, $11M in lobbyist salaries
- 11 confirmed revolving door cases (former governors, senators, DAs → lobbyists)
- AIPAC funding for 7 of 11 MA federal delegation members
- 14 spending anomalies including a 1,559% year-over-year increase
- Campaign contributions to 197 state legislators

**Data sources:**
- MA CTHRU expenditure database (Socrata API)
- OCPF Campaign Finance API
- MA Secretary of State Lobbyist Division
- Federal Election Commission
- OpenSecrets
- TrackAIPAC

All data is from public government databases. All findings represent correlations requiring investigation.
